## Resumo Rápido

O Cortex usa o modelo **W5H** para estruturar memórias:

| Campo | Descrição |
|-------|-----------|
| **WHO** | Quem participou |
| **WHAT** | O que aconteceu |
| **WHY** | Por quê |
| **WHEN** | Quando |
| **WHERE** | Onde (namespace) |
| **HOW** | Resultado |

Para detalhes completos, veja [Modelo de Memória](./concepts/memory-model.md).
